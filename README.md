# OOP Class Hierarchy Project 🧩

This project demonstrates the use of Object Oriented Programming (OOP) principles in PHP by creating a hierarchy of related classes. The goal of the assignment is to practice structuring code using inheritance, properties, methods, and traits while keeping the project organized and well documented.

The project includes a multi-level class hierarchy where a parent class is extended by several child classes, which are then extended further to create deeper levels in the hierarchy. Each class represents a related concept and shares behaviors and properties with other classes through inheritance and reusable code.

This assignment focuses on writing clean, structured PHP code while following proper development practices such as documentation, type hints, and organized project files.

## Features 🎯

Class Hierarchy  
Includes a three-level inheritance structure with a parent class and multiple subclasses.

Object Properties  
Each object contains properties that describe its characteristics.

Reusable Methods  
Shared behaviors are defined in higher-level classes and overridden when necessary.

Trait Implementation  
A trait is used to share functionality between multiple classes without affecting inheritance.

Autoloading  
Uses `spl_autoload_register()` to automatically load class files.

Clean Documentation  
PHPDoc comments are included to document classes, properties, and methods.

## How It Works ⚙️

Class Structure  
The project begins with a parent class that defines shared properties and methods.

Inheritance  
Three child classes extend the parent class to create specialized objects.

Extended Classes  
Four additional classes extend the previous level to create a deeper hierarchy.

Shared Functionality  
A trait is implemented to allow multiple classes to reuse common functionality.

Autoloading  
The `index.php` file uses `spl_autoload_register()` to automatically load class files when they are used.

Execution  
Objects are created in `index.php` and their methods are executed to demonstrate their behaviors.
